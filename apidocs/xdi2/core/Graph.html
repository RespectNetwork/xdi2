Interface Graph

  • All Superinterfaces:
    Comparable<Graph>, Serializable
    All Known Implementing Classes:
    AbstractGraph, DummyGraph, JSONGraph, KeyValueGraph, MemoryGraph, WrappedGraph

    public interface Graph
extends Serializable, Comparable<Graph>
    This interface represents a whole XDI graph. XDI graphs consist of context nodes, relations, and literals. Also, an XDI graph can be expressed as a set of XDI statements.

    markus
