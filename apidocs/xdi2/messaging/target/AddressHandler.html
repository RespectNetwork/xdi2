Interface AddressHandler

  • All Known Subinterfaces:
    ContextHandler
    All Known Implementing Classes:
    AbstractAddressHandler, AbstractContextHandler, GraphContextHandler

    public interface AddressHandler
    An AddressHandler can execute an XDI operation against an address. The AbstractMessagingTarget requests AddressHandler implementations for each address given as part of an operation.
